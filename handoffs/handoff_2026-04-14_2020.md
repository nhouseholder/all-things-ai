- **Headless preview viewport defaults to 0×0** — `mcp__Claude_Preview__preview_screenshot` against a freshly-started server returns a chart-less DOM because `recharts` `<ResponsiveContainer>` measures parent. Always call `preview_resize` first (e.g. `width=1440, height=900`) before screenshotting recharts/anything responsive.
- **`hidden_from_catalog` lives on `subscription_plans`, not `models`** — Earlier audit query mistakenly used `m.hidden_from_catalog` and failed. The `models` table uses `is_active` for visibility filtering.
- **Derived radar values are estimates** — TAU/Arena/SR fallbacks fabricate numbers from related benchmarks. The formulas are conservative (subtract 4-8 points) but they are not measured data. Anyone interpreting the radar at face value (e.g. for procurement decisions) needs to know which axes are derived. That's the v2 ticket above.
