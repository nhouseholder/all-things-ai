## 5. Warnings

- **bash-guard bumps**: two small friction points this session. (1) Bash hook blocks `curl | python3` pipes, so always download with `-o /tmp/x.json` then parse. (2) `wrangler deploy` is blocked if no version file modified in current diff — had to bump v0.16.0 → v0.16.1 to ship the audit fixes even though the logic change was small.
- **Worktree git-push races**: first `git push` after a commit on the worktree branch occasionally reports "won't match the local branch" but actually succeeds. Verify with `git log origin/main` before panicking; re-run push if needed.
- **`api.js` hostname-based base URL**: Claude Preview's proxy makes `window.location.hostname ≠ "localhost"`, so the dev preview hits **prod** worker not local. This is fine for read-only audits but can mask migrations that haven't deployed. Workaround: test against a real localhost tunnel, or check prod API directly with curl.
